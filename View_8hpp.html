#define __LLAMA_DEFINE_FOREACH_BOOL_FUNCTOR (   OP,
  FUNCTOR 
)

Macro that defines two functors for llama::ForEach which apply a boolean operation on a given virtual datum and either another virtual datum or some other type. In the first case the operation is applied if the unique id of the two elements in the datum domain is the same, in the second case the operation is applied to every combination of elements of the virtual datum and the second type. The result is the logical AND combination of all results. So e.g., if some elements are bigger and some are smaller than in the other virtual datum or in the type, for both boolean operations ">" and "<" the functor will return false. For "!=" the operator would return true.

Parameters
OPoperation, e.g. >=
FUNCTORoperation naming used for functor name definition, e.g. if FUNCTOR is "BiggerSameThan", the functors will be named BiggerSameThanBoolFunctor and BiggerSameThanBoolTypeFunctor.
Returns
a bool inside the member variable "result" of the functor
