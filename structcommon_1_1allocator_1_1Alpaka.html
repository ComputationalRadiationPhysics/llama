Detailed Description

template<typename T_DevAcc, typename T_Size>
struct common::allocator::Alpaka< T_DevAcc, T_Size >

Allocator to allocate memory for a llama::View in the llama::Factory using alpaka::mem::buf::Buf in the background. The view created with this allocator can only be used on the host side. For the use of the view on the device see AlpakaMirror.
