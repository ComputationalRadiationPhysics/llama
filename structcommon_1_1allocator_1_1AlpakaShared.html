Detailed Description

template<typename T_Acc, std::size_t T_count, std::size_t T_uniqueID>
struct common::allocator::AlpakaShared< T_Acc, T_count, T_uniqueID >

Allocator to allocate memory for a llama::View in the llama::Factory using alpaka shared memory in the background. The view created with this allocator can only be used on the view side as share memory exists only there.

Template Parameters
T_Accalpaka Acc type of the kernel
T_countAmount of memory needed in byte like needed for llama::allocator::Stack at compile time. In the future this may change and be a run time parameter (at least for dynamic shared memory allocation).
T_uniqueIDat compile time unique ID needed by alpaka, best is to use e.g. __COUNTER__ which is always unique as it increases after each use while preprocessing the code.
See also
