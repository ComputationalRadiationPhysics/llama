Detailed Description

template<typename T, std::size_t T_dim>
struct llama::Array< T, T_dim >

Array class like std::array but suitable for use with offloading devices like GPUs and extended with some (for LLAMA) useful methods.

Template Parameters
Ttype if array elements
T_dimnumber of elements in array
