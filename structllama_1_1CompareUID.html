Detailed Description

template<typename T_DDA, typename T_BaseA, typename T_LocalA, typename T_DDB, typename T_BaseB, typename T_LocalB, typename SFINAE = void>
struct llama::CompareUID< T_DDA, T_BaseA, T_LocalA, T_DDB, T_BaseB, T_LocalB, SFINAE >

Tells whether two coordinates in two datum domains have the same UID.

Template Parameters
T_DDAfirst user domain
T_BaseAFirst part of the coordinate in the first user domain as DatumCoord. This will be used for getting the UID, but not for the comparison.
T_LocalASecond part of the coordinate in the first user domain as DatumCoord. This will be used for the comparison with the second datum domain.
T_DDBsecond user domain
T_BaseBFirst part of the coordinate in the second user domain as DatumCoord. This will be used for getting the UID, but not for the comparison.
T_LocalBSecond part of the coordinate in the second user domain as DatumCoord. This will be used for the comparison with the first datum domain.
