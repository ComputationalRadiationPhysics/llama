Detailed Description

template<typename T_DatumDomain, typename T_DatumCoordOrFirstUID = DatumCoord < >, typename... T_RestUID>
struct llama::ForEach< T_DatumDomain, T_DatumCoordOrFirstUID, T_RestUID >

Can be used to access a given functor for every leaf in a datum domain given as DatumStruct. Basically a helper function to iterate over a datum domain at compile time without the need to recursively iterate yourself. The given functor needs to implement the operator() with two template parameters for the outer and the inner coordinate in the datum domain tree. These coordinates are both a DatumCoord , which can be concatenated to one coordinate with DatumCoord::Cat and used to access the data.

Template Parameters
T_DatumDomainthe datum domain (DatumStruct) to iterate over
T_DatumCoordOrFirstUIDDatumCoord or a UID to address the start node inside the datum domain tree. Will be given to the functor as DatumCoord as first template parameter.
T_RestUID...optional further UIDs for addressing the start node
