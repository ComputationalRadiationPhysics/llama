Detailed Description

template<typename T_Mapping, typename T_BlobType>
struct llama::View< T_Mapping, T_BlobType >

Central LLAMA class holding memory and giving access to it defined by a mapping. Should not be instantiated "by hand" but with a Factory.

Template Parameters
T_Mappingthe mapping of the view
T_BlobTypethe background data type of the raw data, at the moment always an 8 bit type like "unsigned char"
