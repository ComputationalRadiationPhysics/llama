Detailed Description

template<typename T_View, typename BoundDatumDomain, bool OwnView>
struct llama::VirtualDatum< T_View, BoundDatumDomain, OwnView >

Virtual data type returned by View after resolving a user domain coordinate or partially resolving a DatumCoord. A virtual datum does not hold data itself (thus named "virtual"), it just binds enough information (user domain coord and partial datum coord) to retrieve it from a View later. Virtual datums should not be created by the user. They are returned from various access functions in View and VirtualDatum itself.
