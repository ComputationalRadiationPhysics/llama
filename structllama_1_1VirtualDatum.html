Detailed Description

template<typename T_View>
struct llama::VirtualDatum< T_View >

Virtual data type returned by View after resolving user domain address, being "virtual" in that sense that the data of the virtual datum are not part of the struct itself but a helper object to address them in the compile time datum domain
