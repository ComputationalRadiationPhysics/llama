Detailed Description

template<typename T_ParentViewType>
struct llama::VirtualView< T_ParentViewType >

Struct which acts like a View, but shows only a smaller and/or shifted part of a parental (virtual) view. A virtual view does not hold any memory itself but has a reference to the parent view (and its memory).

Template Parameters
T_ParentViewTypetype of the parent, can also be another VirtualView
