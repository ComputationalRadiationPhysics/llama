Detailed Description

template<std::size_t T_alignment = 64u>
struct llama::allocator::SharedPtr< T_alignment >

Allocator to allocate memory for a View in the Factory using std::shared_ptr in the background. Meaning every time the view is copied, the shared_ptr reference count is increased and both copies share the same memory!

Template Parameters
T_alignmentaligment of the memory used by std::shared_ptr, default value is 64
