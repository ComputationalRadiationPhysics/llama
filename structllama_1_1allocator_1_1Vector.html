Detailed Description

template<std::size_t T_alignment = 64u>
struct llama::allocator::Vector< T_alignment >

Allocator to allocate memory for a View in the Factory using std::vector in the background. Meaning every time the view is copied, the whole memory is copied. When the view is moved, the move operator of std::vector is used.

Template Parameters
T_alignmentaligment of the memory used by std::vector, default value is 64
