Detailed Description

template<typename T_ArrayDims, typename T_RecordDim, bool AlignAndPad = false, typename LinearizeArrayDimsFunctor = LinearizeArrayDimsCpp>
struct llama::mapping::AoS< T_ArrayDims, T_RecordDim, AlignAndPad, LinearizeArrayDimsFunctor >

Array of struct mapping. Used to create a View via allocView.

Template Parameters
AlignAndPadIf true, padding bytes are inserted to guarantee that struct members are properly aligned. If false, struct members are tighly packed.
LinearizeArrayDimsFunctorDefines how the array dimensions should be mapped into linear numbers and how big the linear domain gets.
