Detailed Description

template<typename T_UserDomain, typename T_DatumDomain, typename T_LinearizeUserDomainAdressFunctor = LinearizeUserDomainAdress< T_UserDomain::count >, typename T_ExtentUserDomainAdressFunctor = ExtentUserDomainAdress< T_UserDomain::count >>
struct llama::mapping::SoA< T_UserDomain, T_DatumDomain, T_LinearizeUserDomainAdressFunctor, T_ExtentUserDomainAdressFunctor >

Struct of array mapping which can be used for creating a View with a Factory. For the interface details see Factory.

Template Parameters
T_UserDomaintype of the user domain
T_DatumDomaintype of the datum domain
T_LinearizeUserDomainAdressFunctorDefines how the user domain should be linearized, e.g. C like with the last dimension being the "fast" one (LinearizeUserDomainAdress, default) or Fortran like with the first dimension being the "fast" one (LinearizeUserDomainAdressLikeFortran).
T_ExtentUserDomainAdressFunctorDefines how the size of the view shall be created. Should fit for T_LinearizeUserDomainAdressFunctor. Only right now implemented and default value is ExtentUserDomainAdress.
