Detailed Description

template<typename T_UserDomain, typename T_DatumDomain, typename T_TreeOperationList>
struct llama::mapping::tree::Mapping< T_UserDomain, T_DatumDomain, T_TreeOperationList >

Free describable mapping which can be used for creating a View with a Factory. For the interface details see Factory.

Template Parameters
T_UserDomaintype of the user domain
T_DatumDomaintype of the datum domain
T_TreeOperationListthe type of a compile time list (Tuple) used to define the tree mapping
